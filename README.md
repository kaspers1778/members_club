# members_club
The app is deployed on [heroku](https://members-club.herokuapp.com/)  
 
To run application locally follow this steps:  
1. Clone repository by command in cmd  
```git clone https://github.com/kaspers1778/members_club.git```  
1. Open project directory in console or go there by a command  
```cd ./members_club```
1. Run application by IDE or by next command:  
```go run main.go```    
1. Will after build will appear line simular that  
```Server has been started on :8000```  
8000 - this is the port app running at, so the next step is to open in your browser link 
```http://localhost:8000/```
